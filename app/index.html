Printing to your Leitz Icon

This app designs labels at the exact size of your Leitz Icon cartridges (88 mm continuous by default) and renders them at the printer's native 300 dpi, so output prints 1:1.

Getting a label onto the printer

The Leitz Icon uses Leitz's own wireless protocol, which third-party apps can't drive directly. So this app hands off a perfectly-sized file and you send it the last step:

  1. Export PDF / PNG — saves a label at true size. Open it in the official Leitz Icon Software app (or your print workflow) and print at 100% / actual size.
  2. Print — opens the iOS print sheet. Choose AirPrint if your setup supports it, or Save to Files as PDF and import into the Leitz app.

Always print at 100% (no "fit to page" / scaling) so the millimetres stay accurate.

QR codes vs barcodes

  • QR code — scannable by any phone camera. Pick a type (link, Wi-Fi, email, phone, SMS, contact) — a link (URL) opens that page when scanned (like scanning a product in a store app). Raise error correction for scuffed labels, adjust the size, or drop a logo in the centre.
  • Barcode (Code 128) — the 1-D stripes used on shelves and warehouses; great with dedicated inventory-scanner apps, but it only holds a code (no link). Use QR if you'll scan with a phone.

Rotate & length

Use Rotate 90° to run text along the tape, and Auto-fit or Fixed length to make labels exactly as long as you need.

Cartridges

Continuous widths 12–88 mm (up to 2.7 m long) plus die-cut sizes (36×88, 28×88, 26×88, 50×88, 59×102 mm). Your 3.5 in totes/tools labels = the 88 mm continuous cartridge.

Install as an app

In Safari, tap Share → Add to Home Screen. It then runs full-screen and offline, like a native app.
